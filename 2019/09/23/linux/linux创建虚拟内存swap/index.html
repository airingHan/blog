linux创建虚拟内存swap

Sep 23, 2019 | linux

##1.创建一个swapfile用于虚拟内存：
内存大小为物理内存的2倍左右
DD是Linux/UNIX 下的一个非常有用的命令，作用是用指定大小的块拷贝一个文件，并在拷贝的同时进行指定的转换。dd if=/dev/zero of=/root/swapfile bs=16M count=1000

一般是物理内存的2倍，这里是16G

##2.把swapfile改为swap
mkswap /root/swapfile

##3.开启或关闭swap
swapon /root/swapfile
swapoff /root/swapfile

##4.设置开机启动
vi /etc/fstab

/root/swapfile swap swap defaults 0 0
