arthas启动help

1.获取启动参数

java -jar arthas-boot.jar -h

2.都是啥意思

语法为：

–telnet-port 目标jvm监听telnet端口,默认3658
–http-port 目标jvm监听http端口，默认8563
–session-timeout session超时秒数，默认1800秒(30分钟)
–arthas-home the arthas home
–use-version 使用指定版本的arthas
–repo-mirror 使用指定的maven库镜像，例子:center/aliyun或者http的资源地址
–versions 列出本地和远程arthas版本
–use-http 执行使用http下载，默认https
–attach-only Attach target process only, do not connect
-c,–command 命令行执行，多个命令由;分隔
-f,–batch-file 执行脚本文件
–height arthas-client 终端高
–width arthas-client 终端宽
-v,–verbose 详细的，打印调试信息
-h,–help 打印帮助信息
–target-ip 目标虚拟机IP, 默认127.0.0.1
目标进程ID

java
arthas
