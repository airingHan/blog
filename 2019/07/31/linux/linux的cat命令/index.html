linux的cat命令

Jul 31, 2019 | linux

1.名称

连接文件并在标准输出上打印

2.语法

cat [OPTION]… [FILE]…

3.描述

连接文件或标准输入，在标准输出打印。

当没有文件或文件是-，读标准输入。

4.参数

4.1 -A

-A, –show-all

相当于 -vET

4.2 -b

-b, –number-nonblank

由1开始对非空行编号

4.3 -e

相当于 -vE

4.4 -E

-E, –show-ends

在每行末尾显示$

4.5 -n

-n, –number

由1开始对所有行编号

4.6 -s

-s, –squeeze-blank

抑制重复的空输出行 (两个空行只显示一个空行)

4.7 -t

相当于 -vT

4.8 -T

-T, –show-tabs

显示TAB字符为^I

4.9 -v

-v, –show-nonprinting

使用 ^ 和 M- 符号，除了 LFD 和 TAB 之外。

4.10 –help

显示这个帮助并退出

4.11 –version

输出版本信息并退出

5.例子

cat f - g

输出f的内容，然后是标准输入，然后是g的内容

cat

用标准输入到标准输出

6.查看详情

info coreutils ‘cat invocation’
