commit-hashcode是某个 commit 的哈希值，可以用 git log 查看。
因此一般用法是先用 Git log 查看具体commit的哈希值，然后 reset 到那个版本。

说明：
这两个命令都不会对新增文件起作用。因为新增的文件是还未加到 git 的记录里面的，即属于未被 tracked 的状态，所以撤销修改和回退均对其不影响。我们直接手动删除文件就行了。
