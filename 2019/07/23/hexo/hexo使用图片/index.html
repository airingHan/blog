hexo使用图片

1.直接使用，参见markdown使用图片

##2.使用hexo-asset-image插件

https://github.com/xcodebuild/hexo-asset-image

npm install hexo-asset-image –save

修改_config.yml中post_asset_folder 字段为true

生成图片目录位于你markdown文件处例如：

MacGesture2-Publish
├── apppicker.jpg
├── logo.jpg
└── rules.jpg
MacGesture2-Publish.md

会出现这个bug
https://github.com/xcodebuild/hexo-asset-image/issues/47

最新的应该就没问题了，把git上index.js文件覆盖你本地文件，再使用hexo生成就可以了。
