java的图形验证码之kaptcha

概述

什么是kaptcha

https://github.com/penggle/kaptcha

http://code.google.com/p/kaptcha/

kaptcha框架是谷歌开源的一个可高度配置的实用验证码生成工具。

使用简单，很难破解的验证码。

应用场景

在注册、登录、修改密码时有可能是手机号或者邮箱，如果是手机号发送短信验证码，短信运营商一条几分钱，但是如果是恶意用户将给公司带来损失，如果是邮箱，还是会占用服务器资源(连接、CPU、带宽等)，这时候就需要用户先输入一个验证码，防止恶意用户盗刷。

使用

直接参考ruoyi项目。

依赖：

访问http://localhost:8080/captchaImage 获取到img数据 拷贝至 base64转图片的在线工具中查看图片。
