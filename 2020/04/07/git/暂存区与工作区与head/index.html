暂存区与工作区与head

1. 如何让暂存区恢复成和head的一样

git reset HEAD

2.如何让工作区的文件恢复为和暂存区一样

git checkout – 文件名

3.怎样取消暂存区部分文件的更改

git reset HEAD– 文件名[多文件]

4.消除最近的几次提交

git reset –hard 想要变到的commitId

5.开发中临时加塞了紧急的任务怎么处理？

git stash

git status

git stash apply stash还在

git stash pop
