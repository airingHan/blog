linux的chown命令

##1.名称

chown

意思：改变文件属主; 改变属主; 命令; 更改文件拥有者; 改变文件所有权

2.语法

chown [选项]… [所有者] [:[组]] 文件…

chown [选项]… –reference=参考文件 文件…

3.描述

改变文件属主

4.参数

4.1 -c

类似于verbose，但只有在进行更改时才报告

4.2 -f

忽略错误信息

4.3 -R

-R, –recursive
递归地操作文件和目录

4.4 -H

如果命令行参数是指向目录的符号链接，则遍历它

4.5 -L

遍历遇到的每个符号链接到一个目录

4.6 -P

不遍历任何符号链接(默认)

4.7 -v

-v, –verbose

为处理的每个文件输出一个诊断

4.8 –help

显示这个帮助并退出

4.9 –version

输出版本信息并退出

4.10 查看完整文档

info coreutils ‘chown invocation’

5.例子

5.1 修改目录及目录下的文件给用户组a的a1

chown -R a1:a2 /目录
