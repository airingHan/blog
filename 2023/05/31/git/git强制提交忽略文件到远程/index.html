git强制提交忽略文件到远程

问题描述

平时使用git时都会把项目中的target目录屏蔽掉，但是有时候就是需要用这个目录怎么办呢？

1.方法一

之前我都是直接把.gitignore文件中的/target目录去掉，等把需要的这个target文件提交后再把.gitignore改回来，这样比较麻烦。

2.方法二

强制提交忽略的文件到git。

如下，找到文件位置，使用git add -f 文件名 命令来强制提交该文件到git，

并且以后再次提交的时候就已经可以正常提交，不需要强制提交。
