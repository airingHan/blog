linux安装与卸载jdk

1、卸载JDK

1.1、查看已安装的JDK

rpm -qa|grep jdk
返回值：

1.2、删除环境变量

vim /etc/profile
删除JAVA_HOME环境变量

1.3、卸载JDK

rpm -e –nodeps jdk1.8-1.8.0_191-fcs.x86_64

2、安装JDK

2.1、下载jdk安装包rpm类型

https://www.oracle.com/technetwork/java/javase/downloads/index.html

2.2、安装rpm包

rpm -ivh jdk-8u191-linux-x64.rpm
使用1.1查看是否已经安装成功

2.3、配置环境变量

2.3.1、打开配置文件

vim /etc/profile

2.3.2、添加配置信息

export JAVA_HOME=/usr/java/jdk1.8.0_191-amd64
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

2.3.3、立即生效配置文件

source /etc/profile

2.3.4、验证配置信息

echo $JAVA_HOME
echo $PATH
echo $CLASSPATH

3、查看JDK安装路径

3.1、echo $JAVA_HOME

直接命令行输入echo $JAVA_HOME
如果是按照上一步配置了环境变量

3.2、上一步未找到安装路径

3.2.1、which java

定位到执行路径

3.2.2、whereis java

定位文件

3.2.3、直接硬猜

默认安装路径在/usr/java
